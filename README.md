# free-trial-conversion-project


# 📊 Free Trial Conversion Analysis
> 온라인 교육 플랫폼 무료체험 유형(A/B)별 구독 전환율 및 초기 이탈 구조 분석

## 프로젝트 개요

온라인 교육 플랫폼에서 제공하는 두 가지 무료체험 유형(A타입/B타입)이 유료 구독 전환에 미치는 효과를 분석하고, 초기 이탈 구조를 파악하여 **전환율 개선 전략**을 도출한 프로젝트입니다.

| 구분 | 내용 |
|---|---|
| **기간** | 2025.03 ~ 2025.04 |
| **팀 구성** | 3인 (분석 설계 · 통계 검증 담당) |
| **데이터** | 약 7만 명 유저의 66만 건 가입 로그 및 행동 로그 |
| **기술 스택** | Python (pandas, scipy), SQL |

---

## 분석 배경 및 목적

플랫폼은 두 가지 무료체험을 운영하고 있었습니다.

- **A타입**: 특정 콘텐츠 무제한 체험 제공 (2022.01 ~ 2022.11)
- **B타입**: 7일 단위 전체 콘텐츠 체험 제공 (2023.02 ~ 2023.04)

> **핵심 질문**: 무료체험이 유료 전환을 유의미하게 개선하는가? 유형별 전환·이탈 구조는 어떻게 다른가?

---

## 분석 프레임워크

```
분석 대상 유저 정의 → 전환·이탈 퍼널 구축 → 통계 검증 → 전략 제안
```

### 1. 데이터 전처리

- 무료체험 기간과 동일 기간 가입 유저로 분석 대상 한정
- 회원가입 → 무료체험 → 첫 결제 순서 준수 여부 검증
- 중복 결제 건 정제 (194건, 전체의 3.7%)
- 동일 유저의 A/B 타입 중복 체험 → 최종 결제 기준 채택
- 첫 결제 후 레슨 수강 유저만 필터링 (67,663명 → 5,002명)

### 2. 유저 세분화

| 그룹 | A타입 | B타입 |
|---|---|---|
| 체험군 | 1,958명 (48%) | 740명 (78%) |
| 미체험군 | 2,096명 (52%) | 208명 (22%) |

### 3. 핵심 분석 결과

**무료체험의 전환 효과**
- A타입: 미체험 유저 대비 결제 전환율 **2.8배**
- B타입: 미체험 유저 대비 결제 전환율 **33.4배**
- 통계 검증: 무료체험 유저가 미체험 유저보다 유의하게 높은 전환율 (**p < 0.01**)

**초기 이탈 리스크**
- B타입은 미체험 유저 대비 초기 취소율이 **11% 높음** (p = 0.02)
- B타입 취소 그룹은 무료체험 1~7일차 평균 레슨 수가 유의미하게 높음 (p = 0.036)
- 취소 유저의 과잉 소비는 **1~3일차에 집중** (유지 유저 대비 평균 +24.8회), 4일차 이후 격차 소멸

> **핵심 인사이트**: B타입 무료체험은 단기 몰입을 유도하지만, 과도한 학습으로 초기 구독 취소를 증가시킴

## 전략 제안

### 1. 혼합형 체험 운영 (A타입 + B타입)
선택 제공 강화 + 단기 집중 체험으로 전환율 유지 및 초기 취소율 완화

### 2. 무료 체험 기간 축소 (3일)
취소 유저의 과잉 소비가 1~3일차에 집중되므로, 결제 전 과몰입 방지

### 3. 단계별 체험 콘텐츠 설계
- Day 1-2: 인기 강의 3개
- Day 3-4: 맞춤 추천
- Day 5-7: 전체 탐색

→ 기대치를 점진적으로 형성하여 결제 후 이탈 방지
